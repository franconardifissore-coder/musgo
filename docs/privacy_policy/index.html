Política de Privacidad

Última actualización: 10 de abril de 2026

1. Introducción

Musgo es una aplicación web diseñada para ayudarte a organizar tus plantas, controlar el riego y gestionar los espacios de tu casa. Esta Política de Privacidad explica qué datos tratamos, cómo los usamos y qué derechos tienes sobre tu información.

2. Qué datos recopilamos

  • Datos de cuenta, como email, nombre y datos básicos del proveedor de autenticación.
  • Contenido creado por el usuario, como plantas, espacios, historial de riego y preferencias de uso.
  • Imágenes que subas para el reconocimiento de plantas.
  • Datos técnicos básicos, como tipo de dispositivo, navegador y eventos necesarios para operar la aplicación.

3. Para qué usamos tus datos

  • Crear y mantener tu cuenta.
  • Guardar y sincronizar tu jardín entre dispositivos.
  • Permitir funciones de identificación de plantas.
  • Mejorar el producto y resolver incidencias técnicas.

4. Servicios de terceros

Musgo utiliza servicios externos para funcionar correctamente, incluyendo Supabase para autenticación y base de datos, Google para el acceso con OAuth cuando corresponda y APIs externas para funciones como la identificación de plantas.

5. Conservación y seguridad

Conservamos tus datos mientras tu cuenta permanezca activa o mientras sean necesarios para prestar el servicio. Aplicamos medidas razonables para proteger la información, aunque ningún sistema puede garantizar seguridad absoluta.

6. Tus derechos

Puedes solicitar acceso, corrección o eliminación de tus datos. Para ello, puedes escribir a himusgo@gmail.com.

7. Cookies y almacenamiento local

Musgo puede usar cookies técnicas o almacenamiento local para mantener la sesión, guardar preferencias y mejorar el funcionamiento del producto. No vendemos tus datos ni utilizamos rastreo invasivo.

8. Contacto

Si tienes dudas sobre esta política, puedes escribirnos a himusgo@gmail.com.
